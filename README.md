# Pohon Ilmu untuk Ilmu Komputer

Web sederhana untuk menampilkan pohon ilmu dari ilmu komputer. Dibuat sebagai eksperimen penggunaan d3js.

## Menjalankan Sistem

Pastikan anda memiliki NodeJS, dan kemudian jalankan perintah:

    $ npm install -g bower
    $ npm install -g http-server
    $ bower install
    $ http-server -p 8080

Kemudian akses `http://localhost:8080/` pada browser.
